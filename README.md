# The Clean Air Trees Project

Built with Next.js 15 + Tailwind CSS 3.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

| Route | Page |
|-------|------|
| `/` | Homepage |
| `/about` | About Us |
| `/why-trees` | Why Trees Matter |
| `/what-we-do` | What We Do |
| `/projects` | Projects |
| `/blog` | Blog |
| `/donate` | Donate |
| `/volunteer` | Volunteer |
| `/contact` | Contact |
